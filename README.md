## Description
A web app for tracking daily workouts. 
Users can pickup where they left off on their last workout or log a new workout.

Dashboard displays daily stats:
* Minutes spent on workout.
* Weights used in workout.


## Technologies

* Node.js
* MongoDB
* Mongoose
* Express
